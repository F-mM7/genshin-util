# 原神ユーティリティ

原神まわりの計算をブラウザで行うためのツール集です。

公開URL: **https://F-mM7.github.io/genshin-util/**

## できること

5 つの機能を切り替えて使えます。入力した値は自動で保存され、次回開いたときも続きから使えます。

### 樹脂キャパ

「次に原神を開く時刻」までに樹脂がどれだけ回復し、保有上限（200）からどれだけあふれるかを計算します。

- 入力: アクセス予定日時／現在樹脂／濃縮樹脂／消費予定
- 「次の 5 時」「次の水曜 5 時」などのワンクリックプリセット
- 結果: 残り時間、回復量、キャパ余裕、あふれ、可処分樹脂

### 天賦素材

複数キャラ分の「天賦現在 Lv → 目標 Lv」に必要な ★2 / ★3 / ★4 天賦本の総数と、所持数との差分を計算します。

- 行の追加・削除でキャラ数を可変
- 余った下位素材を上位に繰り上げる 3 シナリオで結果を表示
  - 最低（×1）
  - 最高（×2）
  - 期待値
- 必要数を満たせる「充足確率」を併記

### 武器突破素材

武器のレアリティと現在突破段階 → 目標突破段階から、必要素材数と所持数との差分を計算します。

- 入力: クオリティ（★3 / ★4 / ★5 武器）、現在突破、目標突破、★2 〜 ★5 の所持数
- 天賦素材と同じ 3 シナリオ（最低 / 最高 / 期待値）で結果を表示
- 必要数を満たせる「充足確率」を併記

### ダメージ計算

ステータスとバフから評価値・会心バランス・杯トレードオフを計算します。

- 入力: 計算対象ステータス、各種ステータス値、任意のバフカード
- 結果: 評価値、会心率と会心ダメージのバランス、杯選択のトレードオフ

### 聖遺物経験値

現在 Lv → 目標 Lv に必要な聖遺物経験値素材を最小コストで算出します。

- 入力: レアリティ、現在 Lv ／経験値、選択サブステとその値、利用可能な強化素材
- 結果: 必要経験値、素材プラン、強化シミュレーション
- ★5 では強化目標の聖遺物を複数登録して合算プランニング可能
